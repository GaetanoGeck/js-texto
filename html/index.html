TEXTO
Segment descriptions consist of three components, separated by colons: name, type, further details. Each is optional.

Examples:

  • vepNr:String:fill='0'
  • vepNr::fill='0', where the type is guessed from the value
  • vepNr:Date

The name controls the result of the segment:

  • If no name is given, the segment will be ignored
  • If the name #filler is given, the name filler1 will be used the first the, the name filler2 the next time and so on.
